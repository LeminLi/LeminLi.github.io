Leetcode 486.预测赢家

Posted on 2019-02-17 |

两人依次拿，如果Player1赢，则Player1拿的>Player2拿的。把Player1拿的视为”+”，把Player2拿的视为”-“，如果最后结果大于等于0则Player1赢。

  • 该问题没有直接比较一个选手所拿元素的和值，而是把问题转换为两个选手所拿元素的差值。这一点很巧妙，是关键的一步。
